* Describe what the DOM is and an example of how you might interact with it.
The Document Object Model (DOM) is and application programming interface for HTML documents. It allows programs and scripts to dynamically access and update the contents, struture and styles of a document(eg. The webpage/html document). As you would know from the first section of this blog, HTML are plained datas and CSS is the style settings , those are static scripts. Imagine that you have set the hover style in CSS, how is it gonna show on the browser? How would it know the mouse is on the hyper link? How would it know you have click on the link and is time to bring you to the linked page?  DOM will do the job! DOM enables scripts to access document elements and manipulate the datas. It acts like a glue that ties HTML, CSS , and java scripts together. 
DOM is a hierarchical model as shown below:

Every HTML element within the document is an object in the eyes of the DOM (eg. <head>,<body>, <div> etc.) DOM basically describes how the objects are laid out in a tree fashion and each object is called the node. JavaScript works with the objects, it can call methods on document objects thru DOM and interact with them, below is an example of getting objects from DOM:

Document.getElementByTagName(title)

This gets the information from the <title> tag of the HTML element.

* Explain the difference between accessing data from arrays and objects.

Data in arrays is referenced by its position while data in an object is referenced by its defined-properties.

As showing in the picture above, the data stored in an array is call element, each element has a refrenced position in regard to the array length. To access the data within an array, we could use the bracket notation “[ ]” to specify the position of the data. For example arrayName[3], this calls the 4th element inside the array. Note that numbering begins with 0 in programming world, therefore arrayName [n] calls the n+1 th element in that array.

Unlike array, data stored in an object is not refrenced by position. When we create an object, we assign a name to each data set and it is called the property. We use properties to reference data within an object. For example, see the picture above, the names stored in JSobject are the properties, each of the property refers to a set of data. We can use JSobject[Extra Properties] to access the array which contains property “c” and “d”. Or we can use JSobject.Elements to access the array which contains property “0”, “1” & “2”. Generally we use bracket notation “[ ]” to access property has space in its name. And dot notation “.” is use for normal cases. However there are other situations that bracket notation is required. Further information please refer to freeCodeCamp.
